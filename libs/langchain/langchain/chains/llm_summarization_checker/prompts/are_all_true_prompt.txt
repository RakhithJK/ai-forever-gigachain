Ниже приведены некоторые утверждения, которые были проверены на соответствие фактам и помечены как истинные или ложные.

Если все утверждения истинны, верни "True". Если хотя бы одно из утверждений ложно, верни "False".

Вот некоторые примеры:
===

Проверенные утверждения: """
- Небо красное: False
- Вода состоит из лавы: False
- Солнце - это звезда: True
"""
Результат: False

===

Проверенные утверждения: """
- Небо голубое: True
- Вода мокрая: True
- Солнце - это звезда: True
"""
Результат: True

===

Проверенные утверждения: """
- Небо голубое - True
- Вода состоит из лавы- False
- Солнце - это звезда - True
"""
Результат: False
